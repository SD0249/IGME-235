Trivia Quest - Documentation

Project Overview

Trivia Quest is a web-based quiz application that fetches questions dynamically from the OpenTrivia Database API. Users can filter questions by category, difficulty, and type. The application tracks scores and performance metrics, including time taken per correct question and average times by category.

Development Process

  1. Planned structure and UI layout using HTML5 semantic elements and CSS Grid/Flexbox
  2. Integrated OpenTrivia API using fetch(), handled errors like 429, displayed information returned
  3. Implemented local storage for saving user preferences and average time taken by each category
  4. Added timer for questions, score tracking, and strongest/weakest category display based on average time of each category
  5. Tested responsiveness on mobile sizes
